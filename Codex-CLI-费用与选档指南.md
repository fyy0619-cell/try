## 一、价格事实（2026 年 5 月）

### 1.1 个人使用：四种付费路径

| 路径 | 费用（USD/月） | Codex 限额（粗略） | 适合谁 |
|---|---|---|---|
| **ChatGPT Plus** | **$20** | 基准（1x），约 50–150 messages / 5 小时窗口 | 学生、个人开发者、轻中度使用 ⭐ |
| **ChatGPT Pro（$100 新档）** | $100 | Plus 的 5x（促销期间 10x，截至 2026-05-31） | 每天密集跑 `--full-auto`、并发多任务的专业开发者 |
| **ChatGPT Pro（$200 老档）** | $200 | Plus 的 20x（促销期间 25x） | 把 Codex 当主力 Agent 的全职开发者 |
| **纯 API Key 按量付费** | 用多少花多少 | 没有月度上限，但有速率限制 | 用量极少（< 几美元/月）、或想精确控制成本 |

> **重要变化：** 2026-04-02 起 Codex 用量计费从「按消息」改成「**按 token**」（输入 token、缓存 token、输出 token 分别记分）。Plus/Pro 的「额度」现在本质是一个 token 池——**思考时间长 / 跑大上下文 → 消耗更快**。

## 二、用量画像与月度成本估算

按周/天的使用强度分三档，**对号入座**：

| 用法画像 | 每周会话数 | 单次时长 | 是否常用 `--full-auto` | 撞 Plus 上限频率 | 推荐方案 | 实际月支出 |
|---|---|---|---|---|---|---|
| **轻度**：偶尔问问代码、写文档、查报错 | 3–5 次 | 20–40 分钟 | 几乎不 | 极少 | **Plus $20** | **$20** |
| **中度（最常见画像）**：每天 1–2 次，改代码 + 调模型 + 写文档穿插 | 7–14 次 | 30–90 分钟 | 偶尔 | 偶尔（一周 1–2 次） | **Plus $20** | **$20** |
| **重度**：当主力 Agent 用，并发多任务、频繁 `--full-auto`、长跑实验 | 20+ 次 | 1–3 小时 | 经常 | 经常撞 | Pro $100（促销期 10x） | **$100** |

> **学生 / 单人项目 / 副业开发者**绝大多数落在轻度或中度，**Plus 完全够用**。
>
> "我想买 Pro 200 早点上手"几乎从来不是必要——先用 Plus 跑一个月，再判断额度够不够。

## 三、什么时候应该升档？

下面任一条**连续触发两周以上**，再考虑从 Plus 升到 Pro $100：

1. **每天跑超过 3 次"大任务"**（比如让它自动改十几个文件 + 跑测试）
2. **频繁触发"已达上限，等 X 小时再试"**——一周超过 2 次
3. **同时开多个 Codex 实例**（一个写代码、一个跑测试、一个汇总日志）
4. **拿来跑数据集预处理 / 长跑训练脚本**，单次会话 token 量大
5. **业务 SLA 要求**——客户在等，撞限额会直接导致交付延期

满足**4 条以上** + 收入能撑得起，再考虑直接上 Pro $200。

## 四、API Key 按量付费的真实坑

如果你不想付 Plus 的 $20 月费、改用 API Key 按量付费——**听起来更便宜，实际有坑**：

### 4.1 单次成本估算

典型一次 Codex 会话（30 分钟、agent 模式、跑几条命令）大约消耗：

- 输入 token：30k–100k
- 输出 token：5k–20k
- **单次成本：$0.10–$0.30**

按中度用量（每周 10 次会话）粗算：**$4–$12/月**

### 4.2 坑 1：预付余额门槛

- Tier 1 要求最低 **$5 充值**才能开始用
- 必须先有美卡 / 海外虚拟卡 / 国际支付渠道
- 余额用完会突然报 `insufficient_quota`，得回去补

### 4.3 坑 2：大上下文一不小心就爆

- 你让它读一个长文件（比如 5000 行代码）+ 多轮迭代，**单次轻松到 $0.50**
- 不小心开 `--full-auto` 跑半小时，可能就花掉 **$2–5**
- 一个粗心的死循环或长时间 stuck，**单次几十美元**也见过

### 4.4 经验法则

> 每月预期能突破 **$15** 的人都应该直接买 Plus。
>
> 省心、有官方支持、不会因为某次操作不小心烧钱。

API Key 的合理使用场景：**完全偶发**——一个月真的就用 1–3 次，每次都很短。

## 五、对个人用户的选档建议

按性价比从高到低：

1. **首选：买一个月 Plus（$20）试水**。一个月里如果**没**频繁撞限额、**没**觉得不够用——稳定在 Plus，年支出 ≈ $240（约 ¥1,740）。

2. **不要先冲 Pro $100**。除非你试用 Plus 至少**一两个月**后明确感觉额度不够。学生项目、个人项目几乎没人需要这个档。

3. **可以同时保留其他 AI 编码工具**（如 Claude Code、Cursor）。它们长处不同、并用不冲突；不必为了一个砍掉另一个。两个工具加起来月支出大约 $40，相当于一顿外卖钱，换效率提升。

4. **团体方案对个人用户没意义**——除非你已经有团队。等到入职 / 创业有团队场景再考虑 Business（$25/座位）。

5. **API Key 适合什么场景**：
   - 完全偶发使用，月开销预期 < $15
   - 想精确算每个项目的 AI 成本（费用入项目账）
   - 需要在脚本里调用 Codex SDK 而不是交互式用

## 六、中国用户的"隐藏成本"

容易被忽视的三笔实际开销：

| 项 | 成本（每月，估算） | 说明 |
|---|---|---|
| **支付渠道溢价** | 订阅费的 **5–15%** | 国内信用卡 90% 直接付不了 OpenAI。常见走法：Apple 礼品卡 / 海外虚拟卡（WildCard 等）/ 借朋友海外卡。手续费和汇率损耗要算进去 |
| **网络专线** | ¥30–80 | Codex 必须能稳定访问 OpenAI 服务。机场/专线月费一般 30–80 元不等 |
| **额度浪费** | 不可量化 | Plus 的额度按 5 小时窗口刷新，**算不进下一个窗口**。每周只用 2–3 天的人，实际单价比"$20÷30 天"高 |

### 6.1 折算到人民币的真实总成本

按 1 USD ≈ ¥7.25、支付溢价取 8%、专线按 ¥50/月：

| 方案 | 美元月费 | + 8% 充值溢价 | + 网络专线 | **实际人民币月支出** |
|---|---|---|---|---|
| Plus $20 | $20 | 约 ¥158 | + ¥50 | **约 ¥208/月** |
| Pro $100 | $100 | 约 ¥785 | + ¥50 | **约 ¥835/月** |
| Pro $200 | $200 | 约 ¥1,570 | + ¥50 | **约 ¥1,620/月** |
| API Key 中度 | $8 | 约 ¥63 | + ¥50 | **约 ¥113/月**（但有用量飙升风险） |

> **关于支付**：WildCard 等虚拟卡服务有开卡费（约 $9.99）+ 每月维护费（约 $1.99）+ 充值手续费（约 3–5%）。一年下来综合溢价约 8–12%。
>
> 留学生 / 出差人员有海外实体卡的话，能省下虚拟卡费用——但要注意有些银行卡 OpenAI 也会拒。

## 七、想精确估算自己的用量？回答 5 个问题

不想看画像表猜成本？回答下面 5 个数，可以代入 token 池模型算 ±20% 精度的月支出：

1. **每天大概打开几次 codex？**
2. **每次平均聊多久？**（10 分钟 / 30 分钟 / 1+ 小时）
3. **会不会用 `--full-auto`** 让它自动跑命令？（never / sometimes / often）
4. **是不是经常让它读 / 改大文件**？（比如一次性吞几千行代码）
5. **会不会同时开多个 codex 窗口**？

### 7.1 粗略换算公式

按 OpenAI 公开数据 + 实测平均值估算：

```
单次会话平均 token 消耗 ≈
    20k tokens × 会话时长系数（30分钟=1，1小时=2，2小时=4）
  × 自动模式系数（不用=1，偶尔=1.5，经常=3）
  × 大文件系数（不读=1，偶尔=1.5，经常=2.5）

月总 token ≈ 单次 × 每周会话数 × 4.3 周

API Key 月成本 ≈ 月总 token × 0.7（按 70% 输入 + 30% 输出 加权混合费率）/ 1,000,000 × $4 美元
```

### 7.2 三个真实案例

| 用户类型 | 输入 | 月 token | 月成本（API Key 路径） | 月成本（Plus 路径） |
|---|---|---|---|---|
| 硕士生（典型） | 每周 10 次×30 分 + 偶尔 auto + 偶尔大文件 | 约 1.5M | 约 $6 | $20（额度有余） |
| 全职后端开发 | 每天 3 次×1 小时 + 经常 auto + 经常大文件 | 约 25M | 约 $100（且超 Plus 限额） | 需 Pro $100 |
| 副业 Indie Hacker | 每周末 4–6 小时密集开发 | 约 5M | 约 $20 | $20（刚好用满） |
